# Intégration d'un seul fichier CSS
```json
"customStyles": [
  {
     "cssRef": "theme.css", 
     "classRef": "my-theme"
  }
]
```
# Intégration de plusieurs fichiers CSS

```json
"customStyles": [
        {
            "cssRef": "theme.css",
            "classRef": "jo-theme"
        },
        {
            "cssRef": "fonts.css",
            "classRef": "jo-theme"
        },
        {
            "cssRef": "sheets.css",
            "classRef": "jo-theme"
        },
        {
            "cssRef": "objets.css",
            "classRef": "jo-theme"
        },
        {
           "cssRef": "elements.css",
           "classRef": "jo-theme"
        }
    ]
